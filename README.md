# ☕ Morning Ritual Coffeeshop

Welcome to **Morning Ritual**, a cozy little coffeshop website designed with elegance, warmth, and handcrafted style.

## 🌿 About the Project

This site is a fictional coffeeshop presentation page created as a web design project. It features:
- A responsive layout
- Smooth navigation between pages
- A soft, atmospheric style with pastel tones and subtle blur effects
- Clear sections: Homepage, Menu, About Us, and Contact
